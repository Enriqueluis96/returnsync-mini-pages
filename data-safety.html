Privacy Support

Data Safety

Minimal data handling for a narrow Shopify workflow.

ReturnSync Mini is designed to process refund events without storing customer, order, refund, return, sales reversal, or subscription data.

Stored

Merchant settings, session state, billing metadata, aggregated usage counters, and latest technical status.

Not stored

Customer, order, refund, return, sales reversal, subscription records, or full webhook payloads.

AI

Optional, non-destructive, one lightweight call with minimal non-personal context and deterministic fallback.

Webhook flow

  • Authenticate webhook using Shopify app template helpers.
  • Extract only minimum identifiers.
  • Query minimal Shopify Admin GraphQL context only when processing is active.
  • Record non-sensitive technical status.
  • Discard operational context and webhook payload.
